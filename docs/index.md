# Integración con Cuadratura Gaussiana

Este trabajo realiza el cálculo de la integral: 

$$
I = \int_0^\pi \sin(x^2)dx
$$

Esto lo consigue mediante la implementación del método de Cuadratura Gaussina con Polinomios de Legendre, por medio de "cuadrature.py".

## Observaciones

* Esta Integral utiliza la Cuadratura Gaussiana, ya que, al no tener una solución analítica requiere emplear métodos numéricos para obtener la solución.
* Analiza el comportamiento del método mostrando un gráfico de convergencia de los valores calculados, demostrando como el método estima con presición el valor de la Integral.
